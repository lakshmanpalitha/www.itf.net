  1. Unzip the downloaded folder.
  2. Using an FTP client, copy the app , js ,skin and other folders to the Magento software's root directory.

If you are working with a Mac, please make sure you are merging the new folders with Magento and not replacing the Magento directory.

Retina display 22

For Retina display, we need some help. Retina.js is a tiny (only 1.2kB) script that will try and replace any image on your page with a high resolution version found on the server. For it to work properly, you need to name your HiDPI images according to Apple’s “@2x” naming convention. For example, an image in your document called with the tag <img src="images/logo.png"> will be swapped with a high resolution version named logo@2x.png and upload on images folder of you theme.


If you want to add retina display to background images then you need to add those css under bootstrap.css after line number 7007 and your images name should be xyz@2x.png and define images background-size as per example below.
.toggleBtn { background-image: url("../images/expand@2x.png"); background-size: 13px 37px; }
